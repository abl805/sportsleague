ChatGPT Influences

Paste the ChatGPT response directly — the app finds the JSON automatically. Works with a raw array, a wrapped object, multiple ```json``` blocks, or the full chat reply. Influences expire automatically after the specified number of weeks.

Influence types & format 
{
  "influences": [
    {
      "player": "Marcus Jordan",
      "streak": "hot",
      "morale": 8,
      "work_ethic_boost": 0.10,
      "duration_weeks": 2,
      "reason": "Back-to-back 40-pt games, locked in"
    },
    {
      "player": "Devon Price",
      "streak": "cold",
      "morale": -6,
      "loyalty_drop": 0.08,
      "duration_weeks": 1,
      "reason": "Shooting slump, frustrated"
    },
    {
      "team": "MAN",
      "momentum": "hot",
      "locker_room_boost": 5,
      "duration_weeks": 2,
      "reason": "Three-game win streak, energy is high"
    },
    {
      "gm": "MAN",
      "trade_urgency": "high",
      "duration_weeks": 2,
      "reason": "GM eyeing a deal at the deadline"
    }
  ]
}

streak: "hot" (+5 effective skill) or "cold" (−5 effective skill) • morale: positive = boost, negative = penalty • momentum: "hot" (+6 chemistry) or "cold" (−6 chemistry) • trade_urgency: "high" or "low" • All magnitudes are minor by design.
