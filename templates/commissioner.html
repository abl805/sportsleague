ChatGPT Influences

Paste the ChatGPT response directly — the app finds the JSON automatically. Influences expire automatically after the specified number of weeks.

Influence types & format 
{
  "influences": [
    { "player": "First Last", "streak": "hot", "morale": 8, "duration_weeks": 2, "reason": "..." },
    { "player": "First Last", "streak": "cold", "morale": -6, "duration_weeks": 1, "reason": "..." },
    { "team": "ABR", "momentum": "hot", "locker_room_boost": 5, "duration_weeks": 2, "reason": "..." },
    { "gm": "ABR", "trade_urgency": "high", "duration_weeks": 2, "reason": "..." }
  ]
}

streak: "hot" (+5 effective skill) or "cold" (−5 effective skill) • morale: positive = boost, negative = penalty • momentum: "hot" (+6 chemistry) or "cold" (−6 chemistry) • trade_urgency: "high" or "low"
